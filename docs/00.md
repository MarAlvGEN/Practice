# El Reto: "El Analizador de Caracteres"

## Objetivo

Crear una función llamada analizarCadena(texto) que reciba un string
y muestre en consola tres cosas:

    El total de caracteres (sin contar espacios).
    La cantidad de vocales que tiene.
    Cuál es el carácter que más veces se repite.

Conceptos a practicar:

    Bucles (for, for...of o métodos de arrays).
    Condicionales (if/else).
    Estructuras de datos (usar un objeto como "diccionario" o mapa de frecuencias).

Instrucciones

    Crea un archivo logica.js.
    Define la función y pruébala pasando un texto largo. Por ejemplo:
    analizarCadena("nodo de javascript");
    Ejecútalo en tu terminal con: node logica.js.
